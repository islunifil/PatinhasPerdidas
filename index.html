Sobre o projeto

O Patinhas Perdidas é um sistema desenvolvido para incentivar a adoção responsável de animais abandonados e apoiar a causa da proteção animal. O projeto surgiu a partir da realidade do abandono de cães e gatos, um problema que gera sofrimento aos animais e também impacta a sociedade. A proposta é conectar pessoas interessadas em adotar um pet com animais que precisam de um novo lar, além de incentivar doações e conscientizar a população sobre a importância da adoção responsável. Dessa forma, o projeto une tecnologia e solidariedade para contribuir com a redução do abandono e oferecer uma segunda chance aos animais.

Tecnologias utilizadas

  • HTML
  • CSS
  • JavaScript
  • PHP
  • SQLite

Funcionalidades

  • Listagem de animais disponíveis para adoção
  • Visualização dos detalhes de cada animal
  • Formulário para solicitação de adoção
  • Cadastro de doações
  • Integração com banco de dados SQLite
