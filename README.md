# 💛🐾Patinhas Perdidas

## Projeto de Extensão

### Sobre o projeto

O Patinhas Perdidas é um sistema web desenvolvido com o objetivo de incentivar a adoção responsável 
de animais abandonados e facilitar o contato entre pessoas interessadas em adotar. 
O projeto também disponibiliza uma área destinada ao recebimento de doações.
A proposta surgiu diante da realidade do abandono de animais, buscando utilizar a tecnologia 
como ferramenta para promover a conscientização e incentivar ações solidárias.

---

## Objetivos

- Incentivar a adoção responsável de cães e gatos;
- Facilitar o acesso às informações dos animais disponíveis;
- Permitir o envio de solicitações de adoção;
- Disponibilizar uma área para doações;
- Demonstrar a integração entre interface, lógica da aplicação e banco de dados.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- SQLite

---

## Estrutura do projeto

O projeto é composto pelos seguintes arquivos principais:

- `index.php` – Página inicial da aplicação.
- `animal.php` – Exibição dos detalhes dos animais.
- `formulario_adocao.php` – Formulário de solicitação de adoção.
- `doacao.php` – Página destinada às doações.
- `conexao.php` – Responsável pela conexão com o banco de dados SQLite.
- `processa_adocao.php` – Processamento das solicitações de adoção.
- `processa_doacao.php` – Processamento das doações.
- `listar_animais.php` – Recuperação dos dados dos animais cadastrados.

---

## Banco de dados

O sistema utiliza o banco de dados SQLite para armazenar as informações da aplicação.

As principais tabelas são:

- Animal
- Usuário
- Adoção
- Doação

---

## Como executar o projeto

1. Faça o download do repositório ou clone-o utilizando o Git.
2. Abra a pasta do projeto em um editor de código, como o Visual Studio Code.
3. Execute um servidor PHP local, por exemplo:

```bash
php -S localhost:8000
```

4. Acesse:

```
http://localhost:8000/index.php
```
